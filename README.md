# ML_NLP-concepts

This repository contains some frequently used NLP methods.

## Concepts implemented
- Lemmatization and Stemming
- Tokenization
- Removing stopwords
- Building Vocabulary
- N grams
- POS tagging
- Chunking (Noun and verb phrases)
- NER
- Important words in corpus
- Frequency and Rank of words
- Extracting topics using LDA
